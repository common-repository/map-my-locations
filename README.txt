== Description ==

Show your locations as pins on a map, works with Google Maps or Mapbox.

Create a map and display via a shortcode

*   Light and dark theme options
*   Set map pin/marker colour
*   Toggle a side list of location names and addresses
*   Side list ordering
*   Information popup on pin/marker click with item name, description & location
*   Integrates with either Google Maps, or Mapbox - just enter your API key into the pluign settings page

== Installation ==

1. Upload `map-my-locations` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your map API key in the settings screen
